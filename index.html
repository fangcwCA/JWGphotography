About Us

The purpose of this initiative was for us to explore the art of web development and photography. We spent the past few months learning how to code in HTML & CSS which were tools used to build this website without the use of any content management system or website builder like WordPress or Wix. We started from the ground up learning the true fundamentals behind web development, learning through online courses, textbooks, forums, and other open-source materials. We also found leisure in taking photos and posing along with our friends on the McMaster Campus. We've manually uploaded our favourite photos onto this website. Hope you enjoy!
